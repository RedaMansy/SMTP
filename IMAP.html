IMAP

IMAP stands for Internet Message Access Protocol and, as the name implies, it is a protocol used by email clients to retrieve messages from a mail server over a TCP/IP connection.
It was also designed with the aim of allowing management of an email box by email clients. All modern email clients and servers support IMAP.

History

It was designed by Mark Crispin in 1986 as a remote mailbox protocol. There were several versions of IMAP:

  • Original IMAP

    • This was implemented as a Xerox Lisp machine client and a TOPS-20 server.
    Lisp machines are general purpose computers that run Lisp as their main software and programming language.
    They are a good example of a high-level language computer architecture.

  • IMAP2

    • This version introduced the command/response tagging and was the first publicly distributed version.

  • IMAP3

    • Published in 1991, IMAP3 is a rare variant of IMAP and was never accepted by the marketplace.

  • IMAP2bis

    • IMAP2 was extended to now support MIME. MIME is an internet standard that supports other formats such as non-text attachments, text in charaqcter sets other than ASCII and message bodies with several parts. It also added mailbox management funcionality that was absent from IMAP2.

  • IMAP4

    • IMAP2bis was renamed to IMAP4 by the IMAP Working Group

Advantages and Disadvantages

Advantages Disadvantages
Faster response time compared to POP as clients stay connected as long as the UI is active and can download message content on demand, as opposed to being connected to the email server for as long as it takes to download new messages. Complex to maintain, therefore some hosts do not support it
Allows simultaneous access by multiple clients and provdes mechanisms for clients to detect changes made to the mailbox by other connected clients. Uses the host's hard drive space
Provides mechanisms for a client to ask the server to search for messages. This avoids having the client to download every message in their mailbox in order to perform searches. Not available offline
