TCP/IP

TCP/IP stands for Transmission Control Protocol/Internet Protocol.
It is the communication protocol used for the internet and similar networks.
Its main function is to control and manage data transmission over the internet.
It also provides end to end data communication that specifies how data should be packetized, adressed, transmitted, routed and received.
There are four abstraction layers to this functionality. They are:

Link Layer

Contains methods of communication for data that remains within a single network segment

Internet Layer

Provides internetworking between independent networks

Transport Layer

Handles host-to-host communication

Application Layer

Provides process-to-process data exchange for applications

History

TCP/IP was designed in te 1970s DARPA scientists, Bob Kahn and Vint Cerf. Due to their design, they've often been refered to as the fathers of the Internet. While working at the IPTO, Kahn realized the need for the development of an open-architecture network model, where any network could communicatew with any other independent of individual hardware and software configuration. He had four goals in mind:

  • Network Connectivity
    • A network should be able to connect to another through a gateway
  • Distribution
    • There would be no central network
  • Error Recovery
      • Lost packets would be transmitted
  • Black-box Design
      • No changes would have to be made to a network to connect to another
